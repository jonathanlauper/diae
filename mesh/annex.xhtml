FILE
  1. cliquez sur l'icône puis sélectionnez le fichier à charger
  2. modifiez le nom du fichier à charger :
    • c'est le nom qui servira à former le lien vers le document
    • il doit être unique parmi toutes les annexes et ne comporter ni espace ni caractère spécial
  3. envoyez le fichier sur le serveur avec le bouton Enregistrer
  4. répétez l'opération ci-dessus (cliquez sur l'icône à droite du message indiquant le succès pour recommencer)
  5. cliquez sur Terminer lorsque vous aurez fini

Notez que vous ne pouvez télécharger que des documents PDF ou des documents Microsoft Office.
